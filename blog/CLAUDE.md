# chan-blog - Project Documentation

Lume (Deno) を使用して構築された個人ブログプロジェクトです。
キャラクターのイメージに合わせた「アンバー・ナイト（琥珀の夜）」というカスタムテーマを採用しており、イエローとゴールドを主役にした温かみのあるデザインが特徴です。

## プロジェクト概要

- **フレームワーク**: [Lume](https://lume.land/) (Deno 用の静的サイトジェネレーター)
- **ベーステーマ**: `theme-simple-blog` (カスタマイズ済み)
- **デザインコンセプト**: 「脱・紫」。琥珀色とゴールドを基調とし、キャラクターの可愛らしさを引き立てる配色。
- **主な機能**:
  - ダークモード（琥珀の闇テーマ）
  - 全文検索（Pagefind）
  - 自動生成される目次（TOC）
  - タグによるアーカイブ管理
  - RSS / JSON フィード

## 構築と実行

`deno.json` に定義された以下のタスクを使用してプロジェクトを操作します。

- **ビルド**:
  ```bash
  deno task build
  ```
- **ローカルサーバー起動（開発用）**:
  ```bash
  deno task serve
  ```
  デフォルトでは `http://localhost:3000` で起動します。

## 開発規約とスタイルガイド

### 配色ルール（Amber Night Theme）
キャラクターの持つイエロー・ゴールドを最大限に活かすため、以下の配色を徹底してください。

- **メインカラー**: `#B8860B` (深みのあるゴールド)
- **アクセントカラー**: `#FFD54F` (キャラクターのキーカラー)
- **背景色 (Light)**: `#FFF9F0` (クリーム系)
- **背景色 (Dark)**: `#0C0B0A` (深い茶褐色)
- **禁止色**: 紫系統（以前のテーマの残影）は一切使用しないでください。

### ファイル構造
- `_includes/layouts/`: サイトの共通レイアウト（`base.vto` など）。
- `_data.yml`: サイトのメタデータおよび、`extra_head` 内に直接記述されたグローバル CSS スタイル。
- `posts/`: 記事コンテンツ（Markdown 形式）。
- `_config.ts`: Lume のプラグイン設定やカスタムフィルター（`stripTags` など）の定義。

### 特殊なフィルター
- `|> md |> stripTags`: Markdown をレンダリングした後に HTML タグを除去し、プレーンテキストの要約を生成するために使用します（`index.vto` など）。

### アバターとアイコン
- サイト内のアイコン表示は `images/favicon.png` を使用。
- サイズは CSS で `40px !important` に固定され、円形に整形されます。
